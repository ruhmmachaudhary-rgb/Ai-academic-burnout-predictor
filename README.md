# Ai-academic-burnout-predictor
# 🧠 Student Mental Health, Stress & Burnout Prediction System

## Overview
This project is a Machine Learning-based system designed to predict **student stress levels and burnout risk** using academic, behavioral, and psychological features. The system helps in identifying students who may be at risk of mental fatigue, high stress, or burnout.

The project includes:
- Stress level prediction model
- Burnout level prediction model
- Data visualization and model evaluation
- Interactive web application using Streamlit

---

##  Objectives
- To analyze student mental health patterns
- To predict stress and burnout levels using ML models
- To visualize feature importance and model performance
- To deploy an interactive prediction system

---

##  Datasets Used
Two datasets were used in this project:

1. **Academic Stress Dataset**
   - Academic stage
   - Peer pressure
   - Study environment
   - Coping strategies
   - Stress index

2. **Student Mental Health & Burnout Dataset**
   - Age, gender, study hours
   - Sleep and screen time
   - Anxiety and depression scores
   - Academic pressure
   - Burnout level

---

##  Machine Learning Models
- Random Forest Classifier (Stress Prediction)
- Random Forest Classifier (Burnout Prediction)

---

##  Workflow
